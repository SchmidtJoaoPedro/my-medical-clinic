# Sistema de Gerenciamento de Pacientes em C

Este projeto implementa um sistema simples para gerenciamento de pacientes usando arquivos binários. Permite cadastrar, consultar e contar registros armazenados no arquivo pacientes.bin.

### Funcionalidades:

- Cadastro de pacientes

- Consulta por índice

- Exibição da quantidade total de registros

- Editar cadastro de paciente

- Excluir cadastro de paciente

- Ordenar por nome de paciente

- Armazenamento persistente em arquivo binário

### Compatível com Windows, Linux e macOS
